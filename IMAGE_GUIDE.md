## 🌐 Step 3: Where to Download Free Images

Use any of these free image resources:

### Top Recommendations for India Tourism:
1. **Unsplash** - https://unsplash.com/
   - Search: "Ladakh mountains", "Kerala backwaters", etc.
   
2. **Pixabay** - https://pixabay.com/
   - Large collection of Indian destinations
   
3. **Pexels** - https://pexels.com/
   - Free high-quality travel photos
   
4. **Wikimedia Commons** - https://commons.wikimedia.org/
   - Specific Indian monument/place photos

5. **Google Images** (Filter for Creative Commons License)
   - Right-click → "Search Image with Google" → Filter "Creative Commons License"

---

## 📝 Step 4: Download & Save Instructions

1. Visit one of the above websites
2. Search for the destination (e.g., "Ladakh" or "Pangong Lake")
3. Download a high-quality image
4. Save it to the `images` folder with exact filename (e.g., `ladakh.jpg`)
5. **Important:** Keep the exact filename as listed above

---

## 🎨 Image Tips for Best Results

✅ **DO:**
- Use landscape-oriented images (wider than tall)
- Choose clear, bright photos of iconic landmarks
- Use high-resolution images (1000px+ wide)
- Keep file sizes reasonable (< 500KB per image)

❌ **DON'T:**
- Use vertical phone screenshots
- Use very small images (< 600px wide)
- Use images with heavy text/watermarks

## 📞 Troubleshooting

**Images not showing?**
1. Check filename is EXACTLY correct (case-sensitive on some systems)
2. Image file is in `images/` folder (not elsewhere)
3. Refresh your browser (Ctrl+F5)
4. Check file extension is `.jpg` or `.png`

**Want a different image?**
- Simply replace the old file with a new one (keep the same filename)
- Refresh browser to see the change
